Kuesioner: Ketahanan Finansial Generasi Z di Era Ketidakpastian Ekonomi

Instruksi: Pilih jawaban yang paling menggambarkan pendapat Anda pada setiap pernyataan. Skala Likert 1–5: 1 = Sangat Tidak Setuju, 2 = Tidak Setuju, 3 = Ragu-Ragu, 4 = Setuju, 5 = Sangat Setuju.

Bagian A: Literasi Keuangan (X₁)
1. Saya memahami perbedaan antara kebutuhan dan keinginan dalam pengeluaran uang.
2. Saya mengetahui cara menyusun anggaran (budget) keuangan pribadi setiap bulan.
3. Saya memahami pentingnya memiliki dana darurat.
4. Saya memahami risiko jika memiliki terlalu banyak utang.
5. Saya mengetahui pentingnya membayar cicilan atau tagihan tepat waktu.
6. Saya mengetahui perbedaan antara menabung dan berinvestasi.
7. Saya memahami bahwa investasi memiliki risiko dan potensi imbal hasil (return).
8. Saya mengetahui beberapa instrumen investasi dasar (misalnya: deposito, reksa dana, emas).
9. Saya memiliki pengetahuan tentang cara merencanakan keuangan untuk tujuan jangka panjang (misalnya: pendidikan, pernikahan, membeli rumah).
10. Saya memahami pentingnya merencanakan keuangan untuk menghadapi kondisi ekonomi yang tidak pasti di masa depan.
Bagian B: Perilaku Menabung (X₂)
11. Saya secara rutin menyisihkan sebagian uang yang saya miliki untuk ditabung.
12. Saya tetap menabung meskipun jumlah uang yang saya miliki terbatas.
13. Menabung sudah menjadi kebiasaan dalam kehidupan sehari-hari saya.
14. Saya menabung setiap bulan tanpa terputus.
15. Saya tetap berusaha menabung meskipun ada banyak godaan untuk berbelanja.
16. Saya memiliki target atau tujuan tertentu dalam menabung (misalnya: membeli barang tertentu, dana darurat, pendidikan).
17. Saya menabung di lembaga keuangan formal (misalnya: bank, e-wallet, aplikasi keuangan).
18. Saya mencatat atau memantau jumlah tabungan saya secara berkala.
19. Saya lebih mengutamakan menabung daripada pengeluaran untuk hal yang bersifat konsumtif (misalnya: gaya hidup, hiburan).
20. Saat menerima uang (gaji, uang saku, honor, dll.), saya langsung menentukan berapa bagian yang akan ditabung.
Bagian C: Ketahanan Finansial (Y)
21. Saya merasa mampu memenuhi kebutuhan dasar saya (makan, tempat tinggal, transportasi) dengan kondisi keuangan saat ini.
22. Saya jarang mengalami kesulitan keuangan untuk memenuhi kebutuhan wajib setiap bulan.
23. Saya memiliki dana simpanan yang bisa digunakan saat terjadi keadaan darurat (misalnya: sakit, kehilangan pekerjaan, kebutuhan mendesak).
24. Jika terjadi pengeluaran tak terduga, saya masih bisa mengatasinya tanpa berutang berlebihan.
25. Saya merasa kondisi keuangan saya cukup stabil meskipun situasi ekonomi sedang tidak menentu.
26. Perubahan ekonomi (misalnya: kenaikan harga, inflasi) tidak langsung membuat kondisi keuangan saya terguncang.
27. Saya merencanakan keuangan saya untuk menghadapi kemungkinan penurunan pendapatan di masa depan.
28. Saya berusaha mengurangi risiko keuangan (misalnya: menghindari utang konsumtif, memiliki sumber pendapatan cadangan).
29. Saya merasa cukup aman dengan kondisi keuangan saya saat ini.
30. Saya merasa percaya diri dapat menghadapi ketidakpastian ekonomi di masa depan dari sisi keuangan.
